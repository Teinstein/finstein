bsm(): Implementation of Black-Scholes formula for non-dividend paying options

Parameters

  • S (float) – Current asset price

  • K (float) – Strike price of the option

  • T (float) – Time until option expiration

  • r (float) – Risk-free interest rate.

  • sigma (float) – Annualized volatility of the asset’s returns

  • call_value (string) – Takes input as “call”, “put”, “both”

Returns

Price of the options

Return type

float. Returns a dictionary when call_value is set to “both”

ttest(): Calculates T test for single sample, two tailed as well as single. This function works for Paired tests as well as the formula is same.

Parameters

  • X (float) – Sample mean

  • mu (float) – Population mean/ Data mean

  • sd (float) – Standard Deviation

  • n (float) – Sample size

  • alpha (float) – P value Limit/Significance value (default:0.05)

  • two_tail (Boolean) – Enter true for conducting two-tailed test. (default: False)

Returns

[T score, P value, True/False (Test was conclusive or not)]

Return type

list
